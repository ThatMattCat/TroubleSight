# Setup

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
deactivate
```

# Run

You should *not* be in the virtual environment for this, the main.py script handles that.
main.py is a wrapper script that launches the main UI (JSON-to-UI currently).

```
python3 main.py
```

Access at https://localhost:8080
